myFlix

myFlix API is designed to return specific data about movies to the user. This includes:

- A list of movies

- The genre, director, image URL, description about a single movie by title to the user

- Description of the genre by name/title

- Personal Information about the director

myFlix API will also allow users to register, deregister, update their information, and add and remove movies to and from their list of favorites.
