myFlix

Request URL HTTP Method Response
Get list of movies /movies GET A JSON object holding data about all the movies
Get data about movie by title /movies/:Title GET A JSON object holding data about a single movie, containing description, genre, director, image, and if it was featured or not.
Example:
{
"Genre": [
"12345678abc"
],
"Director": [
"abc12345678"
],
"_id": "00000000456",
"Title": "Best Movie Ever",
"Description": "This is the best movie ever.",
"ImagePath": "*insert image path here*",
"Featured": true
}
GET list of genres /genres GET A JSON object holding a list of genres.
GET genre by name /genres/:Name GET A JSON object holding data about genres.
Example:
Genre: Thriller
Description: This would describe the a Thriller.
GET list of directors /directors GET A JSON object holding a list of directors.
GET data about a certain director (bio, birth year, death year) /directors/:Name GET A JSON object holding data about a single director, containing the name, bio, birth year, and death year
Example:
"_id": "345rgtdt5efgd",
"Name": "John Doe",
"Bio": "Bio",
"Born": "01/01/1900",
"Died": "01/01/2000"
Allow new users to register /users POST Show a message that indicates if the registration was successful, if not successful will return error with required info.
Example:
If Username exists:
"harry123 already exists!"
If required info is missing:
"Username" is required, "Password" is required
Allow users to update user info /users/:Username PUT A JSON object with data about what info in the user document should be updated
GET a list of users /users GET A JSON object containing a list of all registered users.
GET a user by Username /users/:Username GET A JSON object holding data about a certain user.
Example:
"_id" : "4356873496uefg98h34985y",
"Username": "harry123",
"Password": "123",
"Email": "harry@gmail.com",
"Birthday" : "2000-01-01"
"FavoriteMovies" : [34985690gvfdg980],
Allow users to add movie to their list of favorites /users/:Username/favorites/:_id POST Return a JSON object with the movie ID added to FavoriteMovies.
Example:
"FavoriteMovies": ["60d90b1b0adf9ce7eb6f7d25"],
"_id": "60dcea4cb4b883b78507187f",
"Username": "johndoe",
"Password": "test123",
"Email": "john@gmail.com",
"Birthday": "2000-01-01T00:00:00.000Z"
Allow users to remove movie from their list of favorites /users/:Username/favorites/:_id DELETE Show a message indicating whether the movie was successfully removed.
Allow users to deregister /users/:Username DELETE Show a message indicating whether the username was successfully removed or not found.
